Listas

indice

INSTRUÇÕES: A tag "ol" serve para fazer uma lista ordenada enquanto "ul" para fazer lista "desordenada"... e "li" são itens de lista. Veja:

< ol >

  1. Coffee
  2. Tea
  3. Milk

< ul >

  • Coffee
  • Tea
  • Milk

A

indice

INSTRUÇÕES: Essa tag serve para fazer hiperlinks e é de fechamento manual. Escreve-se "a" e o atributo "href" que diz para onde o link o enviará. Veja:

Esse link lhe enviará para meu currículo: < a href="https://emanuelsoares.w3spaces.com" target="_top">

Obs.: o atributo "target" pode ter como valores:

  • _blank: abre a página em uma nova janela/aba;
  • _self: abre a página na mesma janela;
  • _parent: abre a página na mesma janela do link;
  • _top: cancela todos os demais frames e abre a nova página no mesmo navegador.

E o valor sem o atributo "target" é "_self"
